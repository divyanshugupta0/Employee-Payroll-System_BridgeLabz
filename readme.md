# Employee Payroll System
This is a simple Employee Payroll System built using Node.js and Express. It allows you to manage employee records, including adding, updating, and deleting employee information.
## Features
- Add new employee records
- Update existing employee records
- Delete employee records
## Installation
1. Clone the repository:
   ```
   git clone
    ```
2. Navigate to the project directory:
    ```
    cd Employpee_Payroll_System_BridgeLabz
    ```
3. Install the dependencies:
    ```
    npm install
    ```
## Usage
1. Start the server:
    ```
    npm start
    ```
2. Open your browser and navigate to `http://localhost:3000` to access the Employee Payroll System.
